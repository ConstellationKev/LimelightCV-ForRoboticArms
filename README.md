# Limelight CV Master Vault

A collection of computer vision pipelines that are built for, but not limited to, the Limelight 3A camera.

# Overview

These algorithms range for contour detection, to color hsv filtering, to a neural network classifier model, and other things. The Limelight 3A is a camera built with its own processor so it can run vision pipelines separate from the main processor to save not only speed but also make it more efficient.

I also have FTC algorithms in there for tracking on multiple occasions/situations and can directly be copy and pasted into your teamcode repository for instant usage if you have the correct parts.
